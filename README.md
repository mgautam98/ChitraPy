# ChitraPy
 
ChitraPy is a digital Image Processing Library in Python.

## Features

* Crop
* Grayscale
* Negative
* Sepia
* Hue
* Salt and Pepper
* Stretch
* Warp
* Rotate
* Invert
* Gaussian blur
* Quick Blur
* Contrast Enhancement (Histogram Equalization)
* Local Contrast Enhancement
* Add Watermark
* Sobel Edge detection
* Halftoning
* Dither
* resize
* sharpen
* Emboss
* Identity
* Outline
* Monochrome
* Zoom
* Contrast adjustment (Histogram sliding)
